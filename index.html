Apple Reminders
for your terminal

A powerful command-line interface for Apple Reminders built with Swift. Create, search, and manage reminders without leaving your terminal.

Features

Everything you need

Complete control over your reminders with a CLI designed for power users.

Complete Management

Create, update, complete, and delete reminders with simple commands.

List Management

Create, rename, and organize reminder lists for different contexts.

Advanced Search

Filter by priority, dates, tags, URLs, alarms, and more.

Tag System

Organize with hashtags like #work #urgent that sync across Apple devices.

JSON Output

Machine-readable output for automation and scripting with jq.

Statistics

Track productivity metrics and completion rates over time.

Installation

Get started in seconds

Choose your preferred installation method.

bash
# Clone and build git clone https://github.com/AungMyoKyaw/apple-reminders-cli.git cd apple-reminders-cli ./install.sh
bash
# Add the tap brew tap AungMyoKyaw/homebrew-tap # Install the CLI brew install AungMyoKyaw/homebrew-tap/reminder

⚠️ On first run, grant Calendar/Reminders permissions in
System Settings → Privacy & Security → Calendars

Usage

Common commands

Get productive immediately with these essential commands.

bash
reminder list
List all reminders
bash
reminder create "Buy groceries" --due-date tomorrow
Create with due date
bash
reminder create "Fix bug #work #urgent" --priority high
Create with tags & priority
bash
reminder complete "Buy groceries"
Mark as complete
bash
reminder search --tag work
Search by tag
bash
reminder list --json | jq '.[0].title'
JSON output with jq
bash
reminder stats
View productivity stats
bash
reminder create-list "New Project"
Create new list
